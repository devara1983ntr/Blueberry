Keyboard Shortcuts

Use these keyboard shortcuts to navigate and control playback:

Space Play/Pause video
↑ ↓ ← → Navigate grids
Enter Select/Play current item
Escape Go back or close modals
F Toggle fullscreen
/ Focus search input

Note: Shortcuts are disabled when typing in input fields.
